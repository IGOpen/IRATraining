This page uses bootstrap 3. FloatThead does not require any css to properly float the table header. However, it does require css to make sure that the floated header does not have a transparent background etc

This page has 2 tables which have the class of table table-striped table-bordered and table table-striped.


Here is the css you need to make them look decent with floatThead:

{% highlight css %} table.floatThead-table { border-top: none; border-bottom: none; background-color: #FFF; } {% endhighlight %}

And the javascript:

{% highlight js %} $("table.table").floatThead() {% endhighlight %}
Note: You might also run into an issue where css from other libs is adding margin to the floated header. See this issue for a resolution.
